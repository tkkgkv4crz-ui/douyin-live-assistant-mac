# 抖音直播助手 for macOS

一款基于 Electron + React + TypeScript 的 macOS 抖音直播助手软件，提供实时弹幕获取、数据看板、自动回复、语音播报、OBS 推流控制等功能。

## 功能特性

- **实时弹幕获取** — 通过 WebSocket 直连抖音弹幕服务器，毫秒级延迟
- **直播间数据看板** — 在线人数、点赞数、弹幕数量、礼物收入实时统计
- **礼物记录** — 完整记录收到的礼物，支持价值统计
- **在线趋势图** — SVG 折线图展示 60 分钟在线人数变化
- **自动回复系统** — 关键词触发自动回复，支持正则表达式和冷却时间
- **语音播报** — TTS 语音播报弹幕、礼物、进场消息
- **弹幕浮窗** — 可置顶显示的弹幕浮窗
- **OBS 推流控制** — 集成 OBS WebSocket
- **多直播间管理** — 同时管理多个直播间
- **暗色模式** — 支持浅色/深色主题切换

## 技术栈

- **Electron 28** + **React 18** + **TypeScript 5**
- **Tailwind CSS 3.4** + **Zustand**
- **Vite 5** + **electron-vite**
- **WebSocket + Protobuf** — 抖音弹幕协议
- **Web Speech API** — TTS 语音播报
- **obs-websocket-js** — OBS 控制

## 安装和运行

### 环境要求

- macOS 12+ (Monterey 或更新版本)
- Node.js 18+
- npm 9+ 或 pnpm 8+

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建应用

```bash
npm run build
npm run dist:mac
```

打包完成后，应用位于 `release/` 目录下。

## 使用指南

### 首次使用

1. 启动应用后，在「直播间」页面点击「添加直播间」
2. 输入抖音直播间链接或房间 ID
3. 点击「连接」按钮开始获取弹幕
4. 切换到「弹幕」页面查看实时弹幕

### 自动回复设置

1. 进入「自动回复」页面
2. 点击「添加规则」
3. 设置关键词和回复内容
4. 启用规则

### OBS 集成

1. 在 OBS 中启用 WebSocket 服务器
2. 在「设置」→「OBS 设置」中输入 OBS 的连接信息
3. 连接后即可在应用中控制推流
